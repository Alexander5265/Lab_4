# Задание 1.

## Задача 10.

### Текст задачи.

Решить задачу на построение нового дерева по заданному (map). 
Дерево содержит целые числа. К каждому числу приписать заданную цифру в
конец.

### Алгоритм решения.

1. Написана рекурсивная функция map, которая проходит по всем узлам и возвращает абсолютно новое бинарное дерево, не изменяя исходное.
2. Добавление цифры в конец числа реализовано без перевода в строку — исходное число умножается на 10, а затем к нему прибавляется нужная цифра.
3. Добавлена проверка знака числа. Для отрицательных значений (например, -5) приписываемая цифра (например, 3) вычитается, чтобы получить корректный результат (-53), а не прибавляется.
4. Входная цифра, которую нужно приписать, нормализуется через взятие по модулю 10 abs (digit % 10). Это гарантирует, что даже если передать число 123, алгоритм возьмет только одну цифру 3.

# Задание 2.

## Задача 10.

### Текст задачи.

Решить задачу на получение заданного результата для указанного дерева (fold). 
Сколько элементов дерева не содержит заданную цифру?

### Алгоритм решения.

1. Реализована обобщенная функция Tree.fold, которая проходит по каждому узлу дерева и прокидывает через них единый счетчик-аккумулятор.
2. Написана хвостовая рекурсивная функция containsDigit, которая последовательно делит число на 10 и проверяет остаток от деления (последнюю цифру), пока не найдет совпадение или пока число не закончится.
3. Предусмотрена корректная проверка ситуации, когда значение самого узла равно 0 (цикл деления для него не сработает стандартным образом).
4. В fold передается стартовое значение счетчика 0. На каждом узле вызывается проверка: если нужной цифры в числе нет, счетчик увеличивается на 1, если есть — счетчик передается дальше без изменений.
5. Функция возвращает целое число — общую сумму всех узлов, удовлетворяющих условию варианта.
